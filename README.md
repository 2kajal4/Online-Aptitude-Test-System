# Online Aptitude Test System

## Description
A web-based Online Aptitude Test System developed using JSP, Servlets, Hibernate, and Oracle Database.
It allows users to register, login, attempt aptitude tests, and view results instantly.

## Features
- User Registration & Login
- Online Aptitude Test
- Automatic Score & Percentage Calculation
- Result & Report Generation
- Hibernate ORM integration
- Oracle Database connectivity
- Secure Logout

## Technologies Used
- Java
- JSP & Servlets
- Hibernate
- Oracle Database
- HTML, CSS, JavaScript
- Apache Tomcat 9

## Tools Used
- Eclipse IDE
- Oracle SQL Developer
- Apache Tomcat
